#### Usage
```
cd .config
git clone git@github.com:ericlake/awesome-config.git awesome
```

#### Wallpaper cron
```
*/5 * * * * DISPLAY=:0.0 feh --randomize --bg-fill $HOME/.config/awesome/    themes/backgrounds >/dev/null 2>&1
```
